Intent-Aware PhishGuard is built with privacy as a core design principle.

## Data Collection
- No personal data is collected
- No browsing history is recorded
- No credentials, OTPs, payment data, or files are captured
- No data is transmitted outside the browser

## Local Processing
All detection, risk evaluation, and insights are processed entirely within the user's browser using Chrome Extension APIs.

## Storage
- Security alerts are stored locally using chrome.storage.local
- A maximum of 20 recent alerts are retained
- Stored data never leaves the device

## Third Parties
- No third-party services are used
- No analytics or tracking libraries
- No external APIs or servers

## Permissions Justification
- storage: store local security alerts
- tabs: deliver warnings to the correct browser tab
- <all_urls>: detect risky intent on visited pages

## Contact
This extension operates fully offline and does not collect user-identifiable information.
